# BluetoothController
+ 아두이노 로봇 제어를 위한 아이폰용 블루투스 컨트롤러

+ 아두이노로 제작할 로봇을 컨트롤하기 위한 아이폰용 컨트롤러 앱 개발용 소스

+ HM-10과 연결 및 메시지 송수신 관련 참조 소스 : https://github.com/0x7fffffff/Core-Bluetooth-Transfer-Demo.git
